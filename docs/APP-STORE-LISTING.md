### Full Description (4000 chars max)
Relix is the universal command center for AI coding agents. Connect Claude Code, Aider, Cline, and every AI coding tool to your phone — with real-time monitoring, push notifications, and one-tap approvals.

🤖 MULTI-TOOL SUPPORT
The only app that works with Claude Code AND Aider AND Cline in one place. More tools coming soon.

💻 MULTI-MACHINE
See all your machines (laptop, server, CI runner) in one dashboard. Know what's running where.

🔒 END-TO-END ENCRYPTED
Your code and AI conversations are encrypted before they leave your machine. We use X25519 + NaCl box encryption — zero-knowledge architecture means we literally cannot read your data.

🔔 PUSH NOTIFICATIONS
Get notified instantly when your AI agent needs approval, hits an error, or finishes a task. No more checking your laptop every 5 minutes.

⚡ ONE-TAP APPROVALS
Approve file writes, command execution, and tool use with a single tap. Review what the AI wants to do and decide — from your phone.

📱 HOW IT WORKS
1. Install relixctl on your machine: brew install relixdev/tap/relixctl
2. Run: relixctl login && relixctl pair
3. Enter the 6-digit code in the app
4. Done — your phone is connected

🆓 FREE TIER

🔓 SELF-HOSTABLE
The relay server is open source (MIT). Run it on your own infrastructure for complete control.

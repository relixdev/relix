# Relix — Demo Video Script

**Duration:** 90 seconds
**Format:** Screen recording (desktop + phone side-by-side)
**Music:** Upbeat, techy, low-key (no vocals)

---

## Scene 1: The Problem (0:00–0:15)

**Visual:** Developer at laptop, Claude Code running. They stand up, walk away. Phone shows nothing.

**Voiceover:**
"You're running Claude Code on a complex refactor. It's going to take a while. You step away — and then it gets stuck. Needs approval. Needs input. But you're not at your desk."

**Text overlay:** "AI agents get stuck when you walk away."

---

## Scene 2: The Solution (0:15–0:30)

**Visual:** Phone notification slides down: "Claude Code needs approval: Write to database.go"

**Voiceover:**
"Relix connects your AI coding agents to your phone. Real push notifications. One-tap approvals. From anywhere."

**Text overlay:** "Relix — Command center for AI coding agents"

---

## Scene 3: Setup (0:30–0:45)

**Visual:** Terminal showing install + pair flow

**Voiceover:**
"Install in 30 seconds. One command to pair with your phone."

**Terminal shows:**
```
$ brew install relixdev/tap/relixctl
$ relixctl login
✓ Logged in as zach
$ relixctl pair
Enter this code in the Relix app: 847291
✓ Paired with Zach's iPhone
```

---

## Scene 4: In Action (0:45–1:10)

**Visual:** Split screen — laptop on left (Claude Code running), phone on right (Relix app)

**Voiceover:**
"See every AI session across every machine. Watch progress in real-time. Approve tool calls with one tap. Send messages directly to your agent."

**Show:**
1. Dashboard with 2 machines, sessions listed
2. Tap into a session — see Claude Code output streaming
3. Approval card appears — tap "Approve"
4. Type a message and send it

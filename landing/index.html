The problem

Your AI agents are stuck when you walk away

AI coding agents need approvals to keep working. When you step away, everything stops.

⏳

Blocked for hours

Claude Code waiting for approval since 3pm — you're at dinner. The session sits idle until you get back to your laptop.

🖥

Too many machines

Seven machines, seven terminal windows you can't see. No single view of what's running, what's blocked, what needs you.

🔒

Privacy concerns

Your session is going through someone else's servers. Enterprise teams can't use tools that route code through third parties.

Features

Everything you need, nothing you don't

Built for developers who run AI coding agents across multiple tools and machines.

🌐

Multi-tool

Claude Code, Aider, Cline, and Cursor support. One app for every AI coding tool you use.

💻

Multi-machine

See all your machines in one dashboard. Laptop, servers, CI — every session at a glance.

🔐

E2E Encrypted

Get notified the instant any agent needs approval. Tap Allow or Deny right from the notification.

How it works

Three steps to never miss an approval again

1

Install

Install relixctl on your machine. One command, no dependencies.

brew install relixdev/tap/relixctl
2

Pair

Open the mobile app, get a pairing code, and link your machine in seconds.

relixctl pair A7X-92K
3

Control

Walk away. Get push notifications. Approve from your phone. Your agents keep running.

Trust

Open source where it matters

The relay and agent are MIT-licensed and on GitHub. Inspect every byte that touches your machine.

relixdev/relay

The WebSocket relay that routes encrypted blobs between your machines and the mobile app. Self-hostable.

MIT License

relixdev/relixctl

The CLI agent that discovers AI coding sessions, bridges them to the relay, and handles encryption.

MIT License
